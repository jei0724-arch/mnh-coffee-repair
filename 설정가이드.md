# M&H 커피머신 AS 접수 사이트 — Firebase 연동 가이드

이 사이트는 3개 페이지로 구성됩니다.

- `index.html` : 홈 (서비스 소개)
- `request.html` : 수리 접수 폼 (폼 제출 → Firestore 저장 + Storage 사진 업로드)
- `gallery.html` / `case-detail.html` : 수리 사례 갤러리 (Firestore `cases` 컬렉션 데이터를 표시, admin.html에서 등록)
- `contact.html` : 문의하기 폼 (Firestore `inquiries` 컬렉션에 저장)
- `confirm.html` : 접수 완료 확인 페이지
- `admin.html` : 관리자 대시보드 (로그인 필요, 접수/문의 리스트 확인 및 상태 변경)
- `firebase-config.js` : Firebase 연결 설정 (이미 값 입력됨)

기능이 실제로 동작하려면 **Firebase 프로젝트**를 만들고 아래 값을 채워야 합니다. 무료 요금제(Spark)로 충분합니다.

## 1. Firebase 프로젝트 생성
1. https://console.firebase.google.com 접속 → "프로젝트 추가"
2. 프로젝트 이름 입력 (예: mnh-coffee-repair) → 생성

## 2. 웹 앱 등록 후 설정값 복사
1. 프로젝트 개요 옆 `</>` (웹) 아이콘 클릭 → 앱 등록
2. 표시되는 `firebaseConfig` 객체 값을 복사
3. `firebase-config.js` 파일의 아래 부분에 붙여넣기

```js
const firebaseConfig = {
  apiKey: "...",
  authDomain: "...",
  projectId: "...",
  storageBucket: "...",
  messagingSenderId: "...",
  appId: "..."
};
```

## 3. Firestore 활성화
1. 왼쪽 메뉴 "Firestore Database" → "데이터베이스 만들기"
2. 위치는 `asia-northeast3(서울)` 권장, 테스트 모드로 시작

테스트 모드는 30일 후 잠기므로, 실제 운영 시 규칙을 아래처럼 설정하세요 (읽기는 인증 필요, 쓰기는 누구나 접수 가능하게):

```
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /repairs/{repairId} {
      allow create: if true;
      allow read: if true;
      allow update, delete: if request.auth != null;
    }
    match /inquiries/{inquiryId} {
      allow create: if true;
      allow read, update, delete: if request.auth != null;
    }
    match /cases/{caseId} {
      allow read: if true;
      allow create, update, delete: if request.auth != null;
    }
  }
}
```

## 4. Storage 활성화 (사진 업로드용)
1. 왼쪽 메뉴 "Storage" → "시작하기" → 테스트 모드로 생성

Storage 규칙 예시:
```
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /repairs/{fileName} {
      allow write: if true;
      allow read: if true;
    }
    match /cases/{fileName} {
      allow read: if true;
      allow write: if request.auth != null;
    }
  }
}
```

## 5. 관리자 계정 만들기 (Authentication)
1. 왼쪽 메뉴 "Authentication" → "시작하기" → 로그인 방법에서 "이메일/비밀번호" 사용 설정
2. "Users" 탭 → "사용자 추가" → 관리자용 이메일/비밀번호 입력
3. 이 계정으로 `admin.html`에 로그인
